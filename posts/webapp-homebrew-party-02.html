Web Application Homebrew Party 2nd


2015 년의 두번째 홈브루 파티가 강남역 토즈에서 있었습니다. 무려 5 시간 동안, 8 명의 참석자분들과 함께 React, React Native 그리고 관련된 여러가지 소재를 다루었습니다. React 에 대해서 각자가 가진 경험과 생각을 공유할 수 있는 있었습니다. React 를 이해할 수 있는 기회가 된 자리였습니다. 회원분들 외에 React 에 관심있는 분들도 참여 해주셨고, 특히 불쑥 부탁드렸는데 준비없이 바로 React Overview 를 해주신 @dalinaum 님께 감사드립니다. 이번에 처음으로 Facebook 에 지속적으로 기록했던 뉴스들을 같이 리뷰하면서 최신 기술 동향에 대해서 이야기 나누었는데요, 상당히 느낌이 좋아서 고정 세션으로 넣기로 했습니다. 특히 Electron 에 대한 반응이 좋아서 Eletron Korea를 개설하기도 했습니다. 이번에 소개된 자료를 아래에 첨부했으니 참고하세요.


  • React 시작해보기 - 한글 튜토리얼 by @dalinaum: React Korea 에서 번역한 튜토리얼로 시작하는 React 을 진행했습니다. WebPagetest 에서 각 Framework/Library 별로 TodoMVC 어플리케이션의 로딩 성능 측정을 보고 rackt 의 Quality React Components 에 대해서 이야기 나누었습니다.
  • React Native 튜토리얼 by @CoderK: React Native 경험기, React Native 의 튜토리얼에 대해서 소개합니다. 튜토리얼을 사용하면서 만든 React Native 메모 를 참고하세요
  • Using Polymer and React on Electron: Electrolysis: by @ragingwind: Electron 에서 Polymer 와 React 사용해보기를 설명했습니다
  • Electron starter kitby @importe: Electron 에서 React 를 사용하는 방법에 대한 스타터킷을 만들고 각 컴포넌트와 사용된 기술들을 설명했습니다
    • React Bootstrap
  • React 와 디3 by @chitacan: React 와 d3 를 아주 잘 사용하는 방법을 찾기 위한 고뇌의 길을 차근차근 짚어주었습니다

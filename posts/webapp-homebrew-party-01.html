Web Application Homebrew Party 1st


2015 을 맞이 해서 2 월 28 일에 강남에서 WAKR 회원들의 홈브루 파티가 열렸습니다. 요사이 자주 언급되는 기술들에 대해서 서로의 관심사를 소개하고 의견을 나누는 자리를 마련했습니다. 저희 회원외에도 관심있는 분들도 오셔서 즐거운 시간을 보냈습니다. 첫 오프모임이라 갖추어지지 않은 부분이 많았음에도 불구하고 서로가 가진 기술에 대한 견해를 교환하면서 서로 영감을 줄 수 있는 시간이였습니다. 이번 홈브루 파티에서는 아래와 같은 주제를 다루었습니다. 관련 자료를 첨부 했으니 참고하세요.


  • Overview Manifest for Web Appplication by @ragingwind: Installable, Service Worker 의 사용을 위한 첫 걸음 Manifest 에 대해서 소개합니다.
  • ServiceWorker: New game changer is coming! by @cwdoh: Manifest 를 이해하셨다면 이제는 Service Worker 에 대해서 알아 봅시다. Offline Web Application 을 위한 새로운 시작
  • Es2015 Simple Overview by @CoderK: ES2015 에서 변경되거나 지원하는 내용을 소개합니다.
  • A React primer by @importe: React 에 반응해봅시다. React 에 기본이 되는 내용을 소개합니다. 주옥같은 슬라이드 소스 도 참고하세요.
